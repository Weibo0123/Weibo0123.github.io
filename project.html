Not home anymore

Project Overview:

Not Home Anymore is a text-based survival game about environmental collapse and resilience.
Set in a scorched farm and the edge of a dying forest, players explore, uncover signs of ecological damage, and make slow, knowledge-driven choices to try to restore the land.
Built in Twine, this project focuses on atmosphere, meaningful consequences, and the quiet challenge of rebuilding life after everything has changed.

Technologies Used:

Twine (SugarCube)
HTML / CSS
JavaScript (for game logic)
